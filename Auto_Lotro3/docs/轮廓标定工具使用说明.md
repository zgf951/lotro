# 轮廓标定工具使用说明

## 功能概述

轮廓标定工具用于小地图箭头方向的模板匹配，主要功能包括：

1. **截取小地图帧** - 从游戏窗口截取包含箭头的图像
2. **提取箭头轮廓** - 支持手动点击或自动提取箭头轮廓
3. **保存模板** - 保存标准轮廓和标准方向（朝北）
4. **实时匹配** - 当前轮廓 vs 模板轮廓，计算旋转角度
5. **显示朝向** - 实时显示当前箭头朝向（角度 + 方向）

## 使用流程

### 1. 打开工具

在主窗口中点击 **"🎯 轮廓标定"** 按钮，打开轮廓标定工具窗口。

### 2. 捕获窗口

1. 点击 **"📷 捕获窗口"** 按钮
2. 将鼠标移到游戏窗口
3. 按 **F1** 键捕获窗口

### 3. 截取图像

点击 **"📸 截一帧"** 按钮，截取当前游戏窗口的图像。

### 4. 提取轮廓

选择提取方式：

#### 自动提取（推荐）
- 选择 **"自动提取"** 单选框
- 点击 **"🔍 提取轮廓"** 按钮
- 系统自动识别箭头轮廓、中心点和箭尖点

#### 手动提取
- 选择 **"手动点击"** 单选框
- 在图像上左键点击选择中心点
- 再点击选择箭尖点

### 5. 设置标准方向

1. 查看提取结果中的角度值
2. 如果箭头当前朝北，点击 **"设为朝北 (0°)"** 按钮
3. 或者手动调整 **"标准方向"** 的数值

### 6. 保存模板

1. 点击 **"💾 保存模板"** 按钮
2. 模板保存到 `G:\map\templates\` 目录
3. 模板文件包含：
   - 轮廓坐标
   - 中心点坐标
   - 箭尖点坐标
   - 标准方向角度

### 7. 加载模板

1. 点击 **"📂 加载模板"** 按钮
2. 选择之前保存的模板文件
3. 模板轮廓会显示在画布上

### 8. 实时匹配

1. 勾选 **"实时自动匹配"** 复选框
2. 或者点击 **"🔄 开始匹配"** 按钮
3. 系统会：
   - 旋转模板轮廓 0~360°
   - 与当前轮廓进行匹配
   - 找到最佳匹配角度
   - 计算当前朝向

### 9. 查看结果

匹配结果会显示在右侧：
- **当前角度**：罗盘方位（0°=北，90°=东，顺时针）
- **方向**：文字方向（北、东北、东、东南、南、西南、西、西北）
- **匹配度**：匹配分数（越小越好）

## 算法说明

### 轮廓提取
- 使用 Canny 边缘检测
- 查找最大轮廓
- 使用 `approxPolyDP` 简化多边形
- 通过最小内角确定箭尖点（比"最远点"方法更准确）

### 角度计算
1. 计算中心点到箭尖点的向量
2. 使用 `atan2` 计算数学角度
3. 转换为罗盘方位（0°=北，顺时针）

### 模板匹配
1. 将模板轮廓旋转 0~360°（步长 5°）
2. 使用 `cv2.matchShapes` 计算匹配分数
3. 找到最佳匹配角度
4. 根据模板的标准方向进行补偿

### 坐标系统
- **数学角度**：X 轴正向为 0°，逆时针
- **罗盘方位**：北为 0°，顺时针
- 转换公式：`bearing = (90 - math_angle + 360) % 360`

## 注意事项

1. **图像质量**：确保截图清晰，箭头轮廓完整
2. **光照条件**：保持游戏内光照条件一致
3. **模板更新**：如果游戏画面变化，需要重新标定
4. **匹配精度**：旋转步长越小，精度越高但速度越慢（默认 5°）

## 常见问题

### Q: 提取不到轮廓？
A: 检查图像质量，确保箭头清晰可见。可以调整 Canny 边缘检测的参数。

### Q: 匹配角度不准确？
A: 1. 确认模板的标准方向设置正确
   2. 尝试减小旋转步长（提高精度）
   3. 检查当前轮廓是否完整

### Q: 如何手动选择点？
A: 选择"手动点击"模式，在图像上左键点击选择中心点，然后点击箭尖点。
